## Projeto Dio - Ferramenta de Apoio IRPF

### Introdução

A **Projeto Dio - Ferramenta de Apoio IRPF** é uma planilha en Excel desenvolvida para servir como facilitador no processo de preenchimento da declaração de ajuste anual de imposto de renda. 

A ferramenta p
ermite que as informações de Rendas Recebidas e Despesas Dedutíveis possam ser registradas durante o decorrer do ano e facilmente transcritas para o *Programa de Ajuste Anual da Receita Federal*.
